üîë Key Differences for E5 Users
Work Tab vs. Web Tab: E5 users with only the free Copilot Chat get the Web Tab (public web data only). The Work Tab, which provides automatic access to your emails, files, Teams chats, and meetings via Microsoft Graph, requires the paid Microsoft 365 Copilot license ($30/user/month). This is the primary feature you're paying for.
Organizational Data Access: Only Microsoft 365 Copilot ($30/mo) has automatic access to your emails, files, Teams chats, and meetings. Copilot Chat (free) requires manual file uploads.
HIPAA Compliance: Microsoft 365 Copilot and Teams Premium support HIPAA when properly configured. Web search queries are NOT covered by HIPAA protections in any version (must be disabled for HIPAA workloads).
Data Protection Level: Consumer Copilot has NO enterprise protections. All work account versions (Copilot Chat, M365 Copilot, Teams Premium) have Enterprise Data Protection covered by the Data Protection Addendum.
Web Search Queries: When web search is enabled, all Copilot versions (except Consumer) send anonymized queries to Bing with NO user/tenant identifiers. These queries are handled separately from your organizational data and are NOT covered by the same DPA/BAA protections.
Model Training: Consumer Copilot may use your data for training. All enterprise versions (Chat, M365 Copilot, Teams Premium) guarantee your prompts and responses are NEVER used to train foundation models.
Audit & Compliance: Only Microsoft 365 Copilot (paid) provides full audit logging, retention policies, and eDiscovery integration. Copilot Chat has basic logging (24hr chat history). Consumer has none.
